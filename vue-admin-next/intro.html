# 基础概念

Vue Admin Next 是一套针对中后台应用场景的企业级 Web 应用脚手架，包含多模块设计、Vue Composition API 深度实践、简洁的用户界面等。

在线预览 - https://openfext.github.io/vue-admin-next

源码仓库 - openfext/vue-admin-next

# 主要特性

  • 开箱即用
    • Vue CLI 4.x
    • Vue Composition API 0.5.x
    • Element UI 2.x
    • Vee Validate 3.x

    • 侧边栏布局方案
